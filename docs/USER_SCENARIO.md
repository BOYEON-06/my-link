# 마이링크 (MyLink) 사용자 시나리오 (User Scenario)

모든 시나리오는 방문자 및 소유자가 특정 목적을 달성하기 위해 수행하는 구체적인 행동(`"사용자는 ___하기 위해 ___한다"`)을 중심으로 정의되었습니다.

---

## 1. 방문자 (Visitor) 시나리오

### 1.1 온라인 프로필 접속 및 상호작용
* 방문자는 크리에이터의 다양한 작업물을 한곳에서 확인하기 위해, 크리에이터의 소셜 미디어 프로필에 등록된 **마이링크 URL(예: domain.com/username)을 클릭하여 접속**한다.
* 방문자는 특정 포트폴리오나 사이트의 상세 내용을 확인하기 위해, 직관적으로 나열된 **링크 블록(버튼) 중 하나를 클릭하여 해당 웹사이트로 이동**한다.
* 방문자는 텍스트 링크 외에 추천 영상 콘텐츠를 바로 감상하기 위해, 리치 미디어 링크 기능으로 **임베드된 동영상의 재생 버튼이나 썸네일을 클릭**한다.

### 1.2 오프라인 QR 코드 접속 (개선 제안 반영)
* 방문자는 오프라인 밋업(세미나, 컨퍼런스) 현장에서 크리에이터의 정보를 빠르게 스마트폰에 저장하기 위해, 배포된 명함이나 팜플렛의 **QR 코드를 스마트폰 카메라로 스캔하여 마이링크 페이지로 진입**한다.

---

## 2. 소유자 (Owner) 관리 시나리오

### 2.1 링크 추가 및 생성
* 소유자는 새로운 포트폴리오 링크를 자신의 페이지에 추가하기 위해, **'새 링크 추가' 버튼을 클릭하여 화면 상단에 즉시 생성되는 빈 링크 블록을 확인**한다.
* 소유자는 링크의 이름과 주소를 빠르고 직관적으로 입력하기 위해, **새로 생성된 블록의 텍스트 영역을 직접 클릭(인라인 편집)하여 내용을 타이핑한 후 엔터를 쳐서 입력**을 완료한다.
* 소유자는 입력한 웹 주소가 올바르게 연결되었는지 직관적으로 확인하기 위해, 인라인 편집 완료 즉시 구글 파비콘 API가 작동하여 **블록 좌측에 해당 사이트의 아이콘(파비콘)이 갱신되는 것을 시각적으로 확인**한다.

### 2.2 링크 수정 및 에러 핸들링 (유효성 검사)
* 소유자는 기존에 등록된 링크의 제목이나 목적지 주소를 바꾸기 위해, 별도의 수정 페이지로 이동하지 않고 **해당 링크 블록의 텍스트를 바로 클릭(인라인 편집)하여 내용을 수정**한다.
* 소유자는 잘못된 URL 형식으로 링크가 작동하지 않는 문제를 방지하기 위해, `https://`가 누락되는 등의 형식 오류 입력 시 **시스템이 붉은색 경고 메시지와 함께 입력을 거부하는 것을 확인하고 올바른 주소 형태로 수정**한다.

### 2.3 링크 위치 정렬 및 시각 효과 관리
* 소유자는 방문자가 가장 먼저 클릭하길 바라는 핵심 링크를 최상단에 배치하기 위해, **해당 링크 블록을 마우스(또는 터치)로 길게 잡고 위아래로 드래그 앤 드롭하여 노출 순서를 변경**한다.
* 소유자는 종료된 특정 이벤트 링크를 삭제하지 않고 목록에서만 일시적으로 감추기 위해, **해당 링크 블록 우측의 '공개 상태 토글 스위치'를 클릭하여 비활성화**한다.
* 소유자는 신규 프로젝트 링크에 방문자의 시선을 즉각적으로 집중시키기 위해, **링크 블록의 '하이라이트 토글'을 켜서 해당 버튼에 반짝임 및 흔들림 애니메이션 효과가 적용되는 것을 확인**한다.
* 소유자는 더 이상 사용할 필요가 없는 이전 기록을 영구적으로 제거하기 위해, **해당 링크 블록의 '삭제 아이콘'을 클릭하여 리스트에서 완전히 지운다.**

### 2.4 프로필 정보 직관적 수정 (인라인)
* 소유자는 본인의 대표 웹 아이덴티티를 갱신하기 위해, **페이지 상단의 아바타(대표 이미지) 영역을 클릭하고 새로운 이미지를 디바이스에서 선택하여 업로드**한다.
* 소유자는 변경된 활동명(닉네임)이나 직업 등의 소개 문구를 수정하기 위해, **기존 프로필의 텍스트 부분(`displayName` 및 `bio`)을 가볍게 탭(클릭)하여 인라인 모드로 진입한 후 새 문구를 덮어쓴다.**

### 2.5 QR 코드 발급 (오프라인 공유 준비)
* 소유자는 컨퍼런스나 사람들과의 대면 미팅 등 오프라인 목적에서 쉽게 마이링크를 배포하기 위해, 관리자 메뉴의 '내 공유 정보'에 들어가 **나의 고유 프로필 주소가 담겨 자동으로 렌더링된 QR 코드 이미지를 디바이스 사진첩에 다운로드**한다.
